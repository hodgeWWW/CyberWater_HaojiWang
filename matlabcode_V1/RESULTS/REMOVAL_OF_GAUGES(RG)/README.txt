This folder contains 9 files, as there are 3 methods: 
(1) Distance-based: "Dist"
(2) Pair-wise-correlation-based: "Corr"
(3) Selection of Graphical Model: "SGM"

For each method, there is a result for 1, 2, and 3 donor gauges.

The format of each table is the following:
Column 1: Rank (order of removal)
Column 2: Gauge Index
Column 3: Nash-Sutcliffe Efficiency (NSE) of inferred daily streamflow.
